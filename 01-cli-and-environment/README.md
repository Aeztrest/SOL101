# 01 - CLI and Environment

## Bu bolumun amaci

- Gelistirme ortamini hizlica dogrulamak
- Kurulumlarin dogru oldugunu komutlarla kontrol etmek

## Gosterilecek komutlar

```bash
# Rust surumu
rustc --version
cargo --version

# Solana CLI surumu
solana --version

# Anchor CLI surumu
anchor --version

# Node.js ve npm surumu
node --version
npm --version
```

## Ogrenci burada ne ogrenir

- Solana gelistirme icin temel araclari
- "Calismiyor" sorununu koddan once ortamda ayirmayi
- Siklikla kullanilan ilk tanilama komutlarini
