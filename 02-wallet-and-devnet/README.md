# 02 - Wallet and Devnet

## Bu bolumun amaci

- Solana config yapisini anlamak
- Devnet'e gecmek
- Wallet ile temel islemleri yapmak

## Gosterilecek komutlar

```bash
# Aktif Solana config bilgisini gosterir
solana config get

# RPC endpoint'i devnet yapar
solana config set --url devnet

# Varsayilan wallet public key gosterir
solana address

# Devnet'ten test SOL ister
solana airdrop 2

# Wallet bakiyesini gosterir
solana balance
```

## Ogrenci burada ne ogrenir

- Solana CLI'nin hangi RPC'ye baglandigini okumayi
- Devnet'te test amacli SOL almayi
- Cuzdan adresi ve bakiye kontrolunu

## Kisa ipucu

Airdrop limiti donemsel degisebilir. Hata alirsaniz daha dusuk miktar deneyin.
