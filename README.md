# SOL101

Bu repo, SOL 101 serisi boyunca üretilen içerikleri düzenli olarak versiyonlamak ve GitHub'a push etmek için kullanılır.

## Amaç
- SOL101 dizisinin tüm içeriklerini tek bir repoda toplamak
- Değişiklik geçmişini takip etmek
- Sürekli güncellemeleri güvenli şekilde push etmek

## Kullanım Akışı
1. Değişiklikleri local'de yap
2. `git add .`
3. `git commit -m "anlamlı bir mesaj"`
4. `git push origin main`
